## 🌍 What is Emowall AI 2.0?

Emowall AI 2.0 is the world's first **multi-generational silent AI safety platform** — built entirely on a phone 📱 using Termux + Acode, from Dubai, for Kerala's children.

While other safety apps wait for a button press, **Emowall acts before the child even realizes danger.**

### 🎯 Mission
> Free child safety for Kerala government school children.  
> No subscription. No compromise. No button needed.

## ⚡ How It Works

```
DETECTION LAYER (Always Active)
├── 👂 Sound Detection — Bad words, crying, hitting sounds
├── ⌚ Smart Watch — Heart rate spike, blood oxygen drop
├── 📱 Phone Sensors — Fall detection, sudden impact
├── 📍 GPS — Location deviation, unsafe zone entry
└── 🧠 7 AI Brains — Analyzing everything silently

        ↓ Threat Detected (Silent — No sound, no screen)

VERIFICATION CHAIN
├── 👨‍👩‍👧 Parent notified → 60 second window
├── 👨‍👩‍👦 Relative notified → 30 second window
├── 🏫 School Principal → 30 second window
└── 🚨 AUTO SOS if no response

        ↓ Auto SOS Triggered

ALERT SYSTEM (Simultaneous)
├── 📍 Live GPS location every 30 seconds
├── 📱 WhatsApp alerts to all contacts
├── 💬 Telegram + Discord notifications
├── 🚔 Kerala Police WhatsApp
├── 🔐 Cyber Cell alert
└── 📞 Auto call to emergency contact
```

## 🔗 Verification Chain

```
Step 1: Parent/Guardian
        └── Notified via WhatsApp + Call
        └── 60 second response window
        └── Reply "SAFE" to cancel

Step 2: Relative (if parent doesn't respond)
        └── Notified via WhatsApp + Call
        └── 30 second response window

Step 3: School Principal
        └── Notified via WhatsApp + Call
        └── 30 second response window

Step 4: AUTO SOS (if no one responds)
        └── Kerala Police WhatsApp: 9497900000
        └── Cyber Cell: 1930
        └── Child Helpline: 1098
        └── Women Helpline: 1091
        └── Live location every 30 seconds
        └── Auto call to police
```

## 🚀 Build & Run

```bash
# Clone
git clone https://github.com/EmoThewall05/Emowall-AI-2.0.git
cd Emowall-AI-2.0

# Install dependencies
flutter pub get

# Run
flutter run

# Build release AAB
flutter build appbundle --release \
  --dart-define=GEMINI_API_KEY=$YOUR_GEMINI_KEY
```
