## 🦋 NEW: Butterfly Guardian

> *"You are safe here. Whatever you share stays between us."*

Butterfly Guardian is Emowall's **emotional wellness AI** — a safe space chat that detects how you feel and responds with healing support.

**How it works:**
- Type how you feel — any language, any emotion
- `EmotionDetector` silently identifies: sad, anxious, angry, lonely, afraid
- `ButterflyGuardian AI` responds with empathy, not scripts
- **Healing Mode** activates for detected distress — purple UI, special badge
- No judgment. No data stored. Just support 💜

**UI:**
- Dark safe space design (`#080c10`)
- Teal bubbles for user · Purple for healing responses
- `PROTECTED` badge always visible
- "Butterfly Guardian • Healing Mode" label on AI responses

```
User types: "I feel scared and alone"
       ↓
EmotionDetector → "fear + loneliness detected"
       ↓
ButterflyGuardian AI → warm, healing, human response
       ↓
Purple bubble + Healing Mode badge shown
```

## 🐾 NEW: Animal Alert

> *"Point. Scan. Stay safe."*

Animal Alert uses your **phone camera + AI** to identify dangerous animals and insects in real time — with immediate first aid and SOS.

**How it works:**
1. Open Animal Alert screen → camera activates
2. Point phone at any animal or insect
3. Tap **🔍 Scan** — AI analyzes in seconds
4. Result shows:
   - Species name + Danger level (HIGH / MEDIUM / LOW)
   - Immediate action steps
   - First Aid instructions
   - Hospital urgency level
   - **ANTIVENOM REQUIRED** warning if needed
5. If attacked → **Long press SOS** → emergency alert fires instantly

**SOS Trigger:**
- Haptic feedback + system alert sound
- Auto sends species + first aid info to emergency contacts
- Connects to `AlertService` → same verification chain as Guardian Mode

```
Camera scan → AI identifies "King Cobra"
       ↓
danger: HIGH · antivenomNeeded: true
       ↓
"ANTIVENOM REQUIRED — Rush to hospital immediately!"
       ↓
Long press SOS → Emergency contacts + location sent
```

Worker: `https://emowall-animal-alert.meradivin.workers.dev`
